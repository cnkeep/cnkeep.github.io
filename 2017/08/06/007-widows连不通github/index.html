github无法ping通

查询github的ip，尝试是否可以连通

这里推荐一个查询ip的工具网站
通过此工具站查询github.com的对应ip，github有多个机房，这里多查询几次，取几个相对稳定的ip，即可。

配置本地DNS

我本机是win10，配置host文件即可:C:\Windows\System32\drivers\etc\hosts, 注意一定要以超级管理员的权限编辑。
